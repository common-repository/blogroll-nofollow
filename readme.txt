This plugin adds the nofollow value to the rel attribute of all blogroll links.

== Description ==

This plugin adds the nofollow value to the rel attribute of all blogroll links.  It is used
mainly for SEO reasons so as not to lose page rank.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `blogroll-nofollow.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy.
